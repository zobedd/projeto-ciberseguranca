# projeto-ciberseguranca
Projeto: Introdução ao Linux para Cibersegurança

Objetivo:
Aprender comandos básicos de Linux e entender o ambiente usado na área de cibersegurança.

O que aprendi:
- Navegação no terminal
- Comandos básicos (ls, pwd, whoami)
- Estrutura de diretórios no Linux

Ambiente:
- Ubuntu (WSL)
- Windows
